Detailed Description

Neuron which computes the hyperbolic tangenst with range [-1,1].

The Tanh function is

\[ f(x)=\tanh(x) = \frac 2 {1+exp^(-2x)}-1 \]

it's derivative can be computed as

\[ f'(x)= 1-f(x)^2 \]

Detailed Description

template<class SearchPointType = RealVector>
class shark::OneNormRegularizer< SearchPointType >

One-norm of the input as an objective function.

The OneNormRegularizer is intended to be used together with other objective functions within a CombinedObjectiveFunction, in order to obtain a more smooth and more sparse solution.

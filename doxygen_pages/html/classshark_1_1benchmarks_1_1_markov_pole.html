Detailed Description

template<class HiddenNeuron, class OutputNeuron>
class shark::benchmarks::MarkovPole< HiddenNeuron, OutputNeuron >

Class for balancing one or two poles on a cart using a fitness function that decreases the longer the pole(s) balance(s). Based on code written by Verena Heidrich-Meisner for the paper

V. Heidrich-Meisner and C. Igel. Neuroevolution strategies for episodic reinforcement learn-ing. Journal of Algorithms, 64(4):152â€“168, 2009.

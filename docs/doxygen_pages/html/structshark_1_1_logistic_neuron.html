Detailed Description

Neuron which computes the Logistic (logistic) function with range [0,1].

The Logistic function is

\[ f(x)=\frac 1 {1+exp^(-x)}\]

it's derivative can be computed as

\[ f'(x)= f(x)(1-f(x)) \]

Detailed Description

template<class InputT>
class shark::HardClusteringModel< InputT >

Model for "hard" clustering.

The HardClusteringModel is based on an Clustering Algorithms object. Given an input, the model outputs the index of the best matching cluster.
See also SoftClusteringModel for general cluster membership.
